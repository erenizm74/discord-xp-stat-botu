75 stardan sonra paylaşılır.

Bot daha %65 bitirildi ana görevini görüyor moderasyon, register ve invite botunuzla birlikte çalıştırabileceğiniz basit hoş bir bot.

Botun içindeki sistemler (%65);

Market sistemi 
Tag aldırma sistemi
